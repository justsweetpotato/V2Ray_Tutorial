# V2Ray_Tutorial
小白也能看得懂的 V2Ray 教程(也许吧XD).
<i>在 2019 年 6 月初这段时间, 梯子出现大面积被封锁的状况, 有网友发现, GFW 会阻断所有非 HTTP 的 TCP 连接. 架设梯子的时候一定要使用 TLS 加密, 1:加密了流量, 2:伪装成了 HTTPS, 是目前我发现最稳的方法. 要是你的梯子不幸已经被封了 IP, 可以套上 CDN(用来做转发)救活, 但是速度会受一定影响.</i>
